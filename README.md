# My [`home-manager`][1] files

## Installation

Install `nix`:

```shell
cd /best/dir/ever
git clone git@github.com:dtgoitia/home-manager.git

ln -s /best/dir/ever/home-manager ~/.config/home-manager
```

## Usage

```shell
home-manager switch
```
